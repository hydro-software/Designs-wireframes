# Naia Points — v2 (HTML prototype) · SUPERSEDED BY naia-v4

> ⚠ **Superseded — see [`naia-v4/`](../naia-v4/).**
>
> v2 was a partial mockup (5 of the 9 planned community pages) built when Naia Community was still being scoped as a stand-alone-feeling experience. After the team review and the [#327 / ADR-001 architecture decision](https://github.com/hydro-software/platform/blob/dev/product-system/architecture-decisions/001-one-integrated-app.md), the canonical mockup is now **`naia-v4/`** — a single integrated mockup covering the existing Naia app plus the Community and Subscription modules, in the actual dark theme, with foldable per-centrale sidebar navigation.
>
> This folder is retained for reference of Community-specific UI patterns (token conversion modal, articles, surveys, market-data cards). Direct stakeholder review and iteration happens in v4.

## Pages built (5 of 9)

- `index.html` — Communauté · Gagner des points
- `rewards.html` — Récompenses with token-conversion modal
- `intelligence.html` — Articles + surveys + webinars
- `market.html` — Mises à jour marché (Belpex / aFRR / rain / production mix)
- `agenda.html` — Events list

These pages remain useful as visual references when porting the Community module patterns into `naia-v4/`.
